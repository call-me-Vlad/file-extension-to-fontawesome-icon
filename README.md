# file-extension-to-fontawesome-icon

A simple array that provides a Font Awesome icon for most common file extensions.

## Usage

in **PHP**

    $ext = "pdf";
    echo $ext_to_fa[$ext];
    // output "fa-file-pdf"

in **JS**

    var ext = "pdf";
    console.log(ext_to_fa[ext]);
    // output "fa-file-pdf"
